# 塔罗随身占卜（微信小程序）

一个可随时随地进行塔罗占卜的微信小程序 MVP。

## 功能

- 单张指引 / 三张时间流 / 三张决策盘 / 关系洞察
- 支持正位和逆位
- 自动生成占卜摘要与行动建议
- 本地历史记录（最多 50 条）

## 目录

- `pages/index`：问题输入与抽牌入口
- `pages/result`：占卜结果展示
- `pages/history`：历史记录管理
- `utils/tarot.js`：牌库、牌阵与解读逻辑

## 使用方式

1. 打开微信开发者工具
2. 选择导入项目，目录选择 `wechat-tarot-miniapp`
3. 填入你自己的小程序 `AppID`（测试可用测试号）
4. 编译运行
